To run locally

after cloning repo

cd in project folder

##Install project dependency 

###From terminal 
python3 on mac
```bash
pip3 install -r requirements.txt
```
other -
```bash
pip install -r requirements.txt
```

##**To run application** 

python3 on mac
```bash
  python3 app.py
```
 for other's -
```
python3 app.py 
```


##Running on 
#####http://127.0.0.1:5000
